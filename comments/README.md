# 注释

**注释**
Stylus支持三种注释: 单行, 多行以及多行缓冲注释

## 单行注释

```stylus
// 单行注释
body
  padding 1px
```

## 多行注释

和常规注释一样. 只有在compress选项唯一用时候才会被输出.

```stylus
/*
* 多行注释
*/
add(a, b)
  a + b
```

## 多行缓冲注释

跟多行注释类似, 不同指出在于开始的时候, 这里是`/*!` 这个相当于告诉Stylus压缩的时候这段无视直接输出.

```stylus
/*!
* 多行缓冲注释
*/
add(a, b)
  a + b
```
